# Issue Tracker: GitHub

Issues for this repository are tracked on GitHub using the official GitHub Issues feature.

## Workflow

- **Creation**: Issues are created using the `gh issue create` command.
- **Reading**: Issues are fetched using `gh issue view` or `gh issue list`.
- **Updates**: Labels and comments are managed via the `gh` CLI.

## AFK Agent Posture

Agents should feel confident creating and updating issues to reflect the current state of a task. When a task is broken down into issues, they should be published in dependency order.
