## What to build
Añadir un estado visual (ej. un pequeño spinner de "Sparkles" o un banner sutil) en el componente `ClassificationFeedback` que indique al usuario cuando el motor de IA está recalculando sugerencias.

## Acceptance criteria
- [ ] Implementar un indicador de carga (loading state) específico para la actualización de sugerencias.
- [ ] Asegurar que el cambio entre sugerencias sea suave y no cause saltos bruscos en la interfaz.

## Blocked by
None - can start immediately
